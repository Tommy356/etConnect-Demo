    ETConnect Reference

    Introduction

    The etConnect framework provides an implementation for Distributerd Objects for iOS and OSX. The framework is available in two versions:

    • ETConnect.framework for iOS (6.0 and higher)

    • ETConnect-OSX.framework for OSX (10.8 and higher)

    This documentation describes the API of the main classes available in etConnect. For a complete description of the framework functionality please refer to the etConnect Programmer’s Reference and the sample applications provided for iOS and OSX.
